Welcome to the Customer API

This API allows you to manage customer information.

Available Endpoints:

  • GET /customers - Get a list of all customers
  • GET /customers/{id} - Get details of a specific customer by ID
  • POST /customers - Add a new customer
  • PUT /customers/{id} - Update an existing customer by ID
  • DELETE /customers/{id} - Delete a customer by ID

All endpoints (except this one) return responses in JSON format.
